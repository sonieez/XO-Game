# XO-Game

## Description 📃
A simple interactive Tic-Tac-Toe Game to play with your friends.

## Features ❌
- Start a new game and make the first move as "X".
- Two-player local multiplayer.
- Match score tracking (first to 3 rounds wins).
- Visual indicator showing whose turn it is.


## Technologies ⭕
- HTML
- CSS
- JavaScript
